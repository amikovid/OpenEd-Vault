# Social Media Automation Priorities

Based on analysis of existing Content Formats, here are the easiest wins for automation:

## Tier 1: Low Hanging Fruit (Start Here)

### 1. Text Heavy/One-Liners (2x/week)
**Why:** Most formulaic, highest engagement
- Simple text on background
- Clear patterns: "Your kid isn't X, school is Y"
- No complex visuals needed
- Can generate 10+ variations easily

**Example formats:**
- iPhone notes style
- White text on black
- Handwritten on graph paper
- Question posts (10-20 words)

### 2. Screenshot Posts (2x/week)  
**Why:** Clear process, easy to template
- Find/create relevant tweet
- Screenshot it
- Drop into Canva template
- Can automate finding + screenshotting

### 3. Memes (3x/week)
**Why:** High frequency, simple formula
- Take education concept
- Match to popular meme format
- Generate variations
- Pick winner

## Tier 2: Medium Effort

### 4. Quote Graphics
**Why:** Pull from existing content
- Extract quotes from podcasts/newsletters
- Apply to template
- Schedule across platforms

### 5. Thought Posts
**Why:** Repurposes one-liners
- Daily features (quotes, wins, success stories)
- Can pull from content bank

## Tier 3: Higher Touch (Semi-Automate)

### 6. Meta Ads
**Why:** Needs creativity but has templates
- Reddit-style posts
- Before/after formats
- Sticky notes
- Requires A/B testing

### 7. Carousels
**Why:** More design work
- Step-by-step guides
- Multi-slide content
- Platform-specific sizing

## Key Automation Patterns

1. **Provocative Statement** → Validation
2. **Question** → Engagement 
3. **Pain Point** → Solution
4. **Before/After** → Transformation

## Next Steps

1. Build one-liner generator (Text Heavy)
2. Set up screenshot pipeline
3. Create meme matcher
4. Test and iterate

The goal: Automate the formulaic stuff so humans can focus on creative strategy and community engagement.
