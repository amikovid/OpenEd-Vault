# Social Media Elements Taxonomy

A unified classification system for all social media elements across platforms, with progressive disclosure to relevant skills and structures.

## Core Elements (~20 Types)

### 1. Text Elements

#### Primary Text
- **Tweet** (280 chars) → `social-content-creation/references/post-structures.md`
- **LinkedIn post** (3000 chars) → `linkedin-content/skill.md`
- **Facebook post** (63,206 chars) → `social-content-creation/references/post-structures.md`
- **Instagram caption** (2200 chars) → `social-content-creation/skill.md`
- **Reddit post/comment** → `.claude/references/social-media-strategy.md`

#### Headlines & Hooks
- **YouTube title** (100 chars) → Check for YouTube title skill
- **Article headline** → `hook-and-headline-writing/skill.md`
- **Email subject** → `hook-and-headline-writing/skill.md`
- **Thread hook** → `hook-and-headline-writing/skill.md`

#### Supporting Text
- **YouTube description** (5000 chars)
- **Pinterest description** (500 chars)
- **Meta description** (160 chars)
- **Alt text** (125 chars)

### 2. Visual Elements

#### Static Images
- **Text-on-background** → Text Heavy format
- **Screenshot post** → Screenshot format
- **Meme** → Memes format
- **Carousel slide** → Carousel format
- **Infographic** → Deep Dives format

#### Video Elements
- **On-screen text** (Reels/TikTok) → `video-caption-creation/skill.md`
- **Captions/subtitles** → `video-caption-creation/skill.md`
- **B-roll with text overlay** → Text on B-Roll format

### 3. Meta Elements

#### CTAs & Engagement
- **Comment prompt** → Engagement frameworks
- **DM trigger** → Comment-to-get strategy
- **Poll/Quiz** → Engagement frameworks
- **Hashtags** → Platform-specific rules

#### Structure Elements
- **Thread structure** → `social-content-creation/references/post-structures.md`
- **Carousel sequence** → Carousel format
- **Story sequence** → Story frameworks

## Progressive Disclosure Map

```
CONTENT TYPE → PLATFORM → ELEMENT → SKILL/REFERENCE
```

### Example Flow:

**One-Liner Post**
1. Type: Text-first content
2. Platforms: FB, IG, LinkedIn, X
3. Elements needed:
   - Primary text (the one-liner)
   - Visual format (text-on-background)
   - Platform caption (if needed)
   - Hashtags (platform-specific)
4. Skills to load:
   - `heavy-hitting-oneliner` (proposed)
   - `social-content-creation` (for proliferation)
   - `meta-ads-creative` (for visual formats)

**Video Post**
1. Type: Video content
2. Platforms: TikTok, Reels, YouTube Shorts
3. Elements needed:
   - On-screen text
   - Hook (first 3 seconds)
   - Caption
   - Hashtags
4. Skills to load:
   - `video-caption-creation`
   - `hook-and-headline-writing`
