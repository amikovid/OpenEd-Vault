## SUMMARY BY FORMAT

| Format | Concepts | Best For |
|--------|----------|----------|
| Text-Only Static | 1-15 | Cold traffic, pattern interrupt, shareability |
| Split-Screen/Us vs Them | 16-25 | Comparisons, solution-aware audience |
| Green Screen Commentary | 26-32 | Education, myth-busting, longer engagement |
| Ad Reaction | 33-38 | Addressing objections, extending reach |
| Educational Micro-Series | 39-44 | Deep engagement, retargeting sequences |
| Partnership Ad Scripts | 45-50 | Trust building, lower CPA, creator channels |

## WHAT'S DIFFERENT FROM V1

1. **Text-Only Static** - Brand new format, pure copy, no visuals needed
2. **Split-Screen** - Visual comparison format not in V1
3. **Green Screen** - Commentary/education format not in V1
4. **Ad Reaction** - Meta-engagement format not in V1
5. **Micro-Series** - Serialized storytelling not in V1
6. **Partnership Scripts** - Written for creator handles, not brand

**Zero overlap with V1 formats** (Notes App, Tweet screenshots, Reddit posts, Meme templates, Testimonial cards, Carousels, UGC talking heads)

- Rigid vs Ready
- Busywork vs Brilliance
