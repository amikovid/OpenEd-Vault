# Open Ed Podcasts

**Ongoing workflow** - Weekly podcast episodes with full asset production.

---

## Quick Start

1. Use skill: `podcast-production` (4-checkpoint workflow)
2. Active episodes in dated folders
3. Archived episodes in `Archived Podcasts/`

---

## File Structure

```
Open Ed Podcasts/
├── PROJECT.md              ← You are here
├── [Guest Name]/           ← Active episode folders
│   ├── transcript.md       ← Raw/polished transcript
│   ├── blog-post.md        ← Companion blog post
│   ├── clips/              ← Video clips
│   └── social/             ← Social media assets
└── Archived Podcasts/      ← Completed episodes
```

---

## Workflow (4 Checkpoints)

1. **CP1: Strategic** - Identify hooks, audience angles
2. **CP2: Clips** - Extract 3-5 standalone clips with captions
3. **CP3: YouTube** - Titles, descriptions, thumbnails
4. **CP4: Written** - Blog post, newsletter integration
