## How to Run Reports

### All Weekly Reports (Recommended)
```bash
cd "Studio/SEO Content Production/seomachine"
./tools/weekly_reports.sh
```
This generates both SEO and Social reports in `Analytics & Attribution/reports/`

### Individual Reports
```bash
cd "Studio/SEO Content Production/seomachine"

# SEO Report
python3 tools/weekly_seo_report.py --domain opened.co --output markdown --save "../../../Analytics & Attribution/reports/weekly-seo-$(date +%Y-%m-%d).md"

# Social Report
python3 tools/weekly_social_report.py --output markdown --save "../../../Analytics & Attribution/reports/weekly-social-$(date +%Y-%m-%d).md"
```

### Quick Stats Check

cd "Studio/SEO Content Production/seomachine"
