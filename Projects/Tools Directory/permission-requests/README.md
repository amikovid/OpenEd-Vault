# Permission Request Drafts

These files contain the **full review content** for each tool, formatted for sending to teachers for approval before publishing.

## Content Structure (Per Fred's Feedback)

Each review includes:
1. **Quick Verdict** - One paragraph summary (will appear above Teacher's Take on final page)
2. **Teacher's Take** - The teacher's narrative with woven colleague quotes
3. **Who [Tool] Is Best For** - Bullet list with tool name in header
4. **Who [Tool] May Not Fit** - Bullet list with tool name in header

## AI-Tells Checklist (Applied)

- [x] No correlative constructions ("X isn't Y - it's Z")
- [x] No setup phrases ("The best part?", "Here's the thing")
- [x] No staccato fragments ("No fluff. No filler.")
- [x] No triple adjectives
- [x] No cliché openers ("In today's...")
- [x] No thesaurus abuse (utilize, comprehensive, crucial, leverage)
- [x] Hyphens with spaces - like this - not em dashes
- [x] No emojis
