## Research: YouTube Ads vs Meta Ads

### Questions to Answer

1. **Audience:** Where are homeschool parents? YouTube or Meta?
2. **Cost:** CPM/CPC comparison for education/parenting niche
3. **Format:** What ad formats work on each platform?
4. **Intent:** YouTube = search intent vs Meta = interruption. Which fits us?
5. **Retargeting:** Can we retarget website visitors on YouTube? How?
6. **Attribution:** How do we track YouTube ad ‚Üí application?

### Resources to Check

- Google Ads (YouTube) documentation
- Meta Ads Manager benchmarks
- Competitor analysis: Are other ESA/homeschool orgs on YouTube?
- OpenEd historical data: Any past YouTube ad experiments?

### Output

1-2 page summary with recommendation: Test YouTube ads in Feb? If yes, what budget and format?

This sprint is about **finding what works**, not perfection.

- Ship fast
- Measure everything
