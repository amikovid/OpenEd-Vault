## Phase 1: Internal Beta (Weeks 1-2)
**Objective:** Seed content + find power users

### Week 1: Staff Outreach
- [ ] Email all-staff announcing Curriculove beta
- [ ] Create simple Loom walkthrough (2-3 min)
- [ ] Set up Slack channel #curriculove-feedback
- [ ] Target: 20+ staff members to try the quiz + submit 1 review

### Week 2: Power User Identification
- [ ] Track who submits multiple reviews (>3)
- [ ] Note quality of reviews (detailed, helpful, authentic)
- [ ] Identify **10 power users** who:
  - Have broad curriculum experience
  - Write compelling reviews

## Phase 2: Content Moat Building (Weeks 3-4)
**Objective:** Hit critical mass for SEO value

### Power User Program
- [ ] Weekly 15-min "review sprint" calls with power users
- [ ] Goal: Each power user submits 5-10 reviews
- [ ] Create review prompts/templates to make it easy
- [ ] **Incentives:**
  - OpenEd swag for participants
  - **Top 3 reviewers get public credit** (name on site, "Founding Reviewer" badge)
  - Gift card raffle for active contributors

### Existing Reviews Migration
- [ ] **Mass Slack message** to teachers who have written partial reviews already
- [ ] Ask them to approve their official review copy on the curriculum page itself
- [ ] Make approval flow dead simple (one-click confirm)

### SEO Foundation
- [ ] Ensure each curriculum page has unique URL
- [ ] Add structured data (Review schema)
- [ ] Create "Best of" category landing pages:
  - Best Math Curriculum for Visual Learners
  - Best History Curriculum for Secular Families
  - Best Science for Elementary

**Exit criteria:** 150+ reviews, 10+ curricula with 3+ reviews each

## Phase 3: Soft Launch (Weeks 5-6)
**Objective:** Test with real homeschool parents

### Limited External Release
- [ ] Share with OpenEd parent community (existing customers)
- [ ] Post in 2-3 homeschool Facebook groups (with permission)
- [ ] Monitor:
  - Quiz completion rate
  - Review submission rate
  - Email capture rate
  - Feedback/bugs

### Content Marketing Start
- [ ] Draft 3 blog posts featuring Curriculove data:
  - "What 100 Homeschool Parents Say About [Popular Curriculum]"
  - "The Most Loved (and Hated) Curricula of 2026"
  - "How Real Families Mix-and-Match Curriculum"

**Exit criteria:** 50+ external users, <5% bounce on quiz
